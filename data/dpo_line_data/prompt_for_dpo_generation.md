
### 任务说明

你是一个数据增强专家。我会给你一批中科大软件学院的 QA 问答数据（instruction + output），请你对每一条数据进行**变体改写**，生成用于 DPO（Direct Preference Optimization）训练的偏好数据。

### 变体规则

**对 instruction（问题）的改写要求：**
1. 保持问题的核心意图不变，但换一种问法/角度/表述
2. 可以采用以下变体方式（随机选择）：
   - 同义改写：换词换句式，如"怎么样" → "如何评价"、"有哪些" → "能列举一下吗"
   - 角度转换：从不同身份提问，如"新生视角"、"准备考研的同学"、"已经在实习的同学"
   - 具体化：把宽泛的问题变具体，如"课程怎么样" → "这门课的考试难不难"
   - 追问式：基于原问题延伸，如"除了XX之外，还有什么建议吗"
   - 口语化/正式化：调整语气风格
3. **禁止**：完全照抄原问题、改得面目全非导致答案对不上

**对 chosen（好答案）的改写要求：**
1. 保留原始答案中的**所有关键事实信息**（人名、课程名、数字、链接等不能改）
2. 可以调整：
   - 表述方式和句式
   - 段落结构和排列顺序
   - 补充合理的过渡句或总结句
   - 适当调整详略（但总长度保持在 100-300 字）
3. **禁止**：删除关键事实、编造新信息、改变原意

### 输出格式

严格按照以下 JSON 数组格式输出，每条数据包含 4 个字段：

```json
[
  {
    "instruction": "改写后的问题",
    "input": "",
    "chosen": "改写后的高质量答案（100-300字）",
    "rejected": ""
  }
]
```
